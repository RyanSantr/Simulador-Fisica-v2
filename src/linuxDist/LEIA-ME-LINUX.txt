SIMULAFISICA 3D - VERSAO LINUX

Como abrir sem IntelliJ ou VS Code:

1. Extraia a pasta SimulaFisica3D-Linux.
2. Clique duas vezes em SimulaFisica3D.desktop ou SimulaFisica3D.sh.
3. Se o Linux perguntar, permita executar como programa.

Se nao abrir por falta de Java:

- Instale Java 21 no Linux; ou
- Coloque um JDK 21 Linux portatil dentro desta pasta com o nome runtime.

O caminho esperado para Java portatil e:

SimulaFisica3D-Linux/runtime/bin/java

O pacote ja inclui as bibliotecas JavaFX para Linux. Nao precisa abrir IDE.
